    Week1

    1.1 Natural Language Content Analysis

    自然语言处理(NLP)分几个步骤

    • Lexical analysis 词汇分析（almost 97%）
    • Syntactic Analysis 语法分析（almost 90%)
    • Semantic analysis 语义分析
    • Pragmatic analysis 实用分析

    Ambiguity是最大问题，同时common sense是被需要的
    Inference还不大好，由于推断和不确定性，这也是AI的挑战

    NLP techniques must be general, robust, and efficent

    • Bag of words 在 modern search engine 中依旧是主流
    • Deeper NLP is needed for future search engine

    1.2 Text Access

    - Push vs. Pull

    • Pull: serch engines
      • User take initiative
      • Ad hoc information need
    • Push: recommender systems
      • System* take initiative
      • Stabel information need

    - Pull mode: Querying vs. Browsing

    • Querying
      • User enter a keyword
      • sys return relevant info
      • work well when user knows what words to use
    • Browing
      • User navigates into relevant info
      • …

    Today we have very good support for query not browsing

    1.3 Text Retrieval /rɪ`trivl/ Problem

    - 什么是Text Retrieval(TR)
    收集文件，用户给问题，反馈信息给user，info retrieval(IR)范围更广
    在工业界叫做search techology

    - Text Retrieval vs. DTbase Retrieval
    info: 前者是unstructured
    query: 前者是ambiguous, 后者complete specification
    Answer: 前者是relevant doc, 后者matched records**
    sum: 前者是 empirically defined problem

    - Doc selection vs. Doc ranking
    前者, 0 or 1, absolute relevance
    后者, 大于theta, relative relevance, cutoff 是 theta

    remove redundancy

    Summary

    • Text retrieval 是 经验 问题，其算法需要被用户决定
    • Doc ranking 是更有效的去 评估结果 和 避免绝对相关性(0 OR 1这种排序)
    • 最大的问题!: 如何设计ranking function，f(q,d) (q: query, d: document)

    1.4 Overview of Text Retrieval Methods

    - Similarity-based models: Vector space model
    - Probabilistic models: Language model
    - Probabilistic inference model:
    - Axiomatic model: f(q,d) must satisfy a set of constraints
    这些models在相似的变量下，都是给出的相似的ranking function

    f(q=”presidential campaign news”, d)
    g(“presidential”, d)
    Term Frequency(TF): c(“presidential”, d)
    Doc Frequency: df(“presidential”) //总之就与word出现的次数，和Doc的长度都有关
    P(“presidential”|collection)

    Models

    • Pivoted length normalizatoin
    • BM25 (most popular)
    • Query likelihood
    • PL2

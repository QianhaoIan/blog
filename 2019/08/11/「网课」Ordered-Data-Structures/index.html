    Week1

    1.1 Arrays

    特点：

    1. Array里面全是相同的类型数据(list一样)
    2. 大小确定，不能改
    3. 在内存里面是连续的

    1 byte 是 8 bits, 1 int 是 4 bytes

    int array[3] = {1, 2, 3};  // definition

    1.2 Linked List

    head pointer 代表开始
    nullptr 代表最后

    1.3 Run Time

    Big O big O notation: express the largest term w/t constants
    logarithm 对数 r = lg(n)

    Array Resize Strategy
    array加项目，有2种方法（加固定长度 + 原本长度加倍）
    - Strategy #1: grow array by +2 each time O(n^2)
    - Strategy #2: double array each time O(n)

    amortized runningt time
    对于 Strategy #2 是 O(1)
    可以理解为，进行一次扩容+copy操作时，进行了n次操作，但是这时使得我们可以insert来n个items
    所以平均加下，每次平均下来是 O(1)

    Big-O natation
    O(1), constant time
    O(n), linear time
    O(n^2), polynomial time

    1.4 Array and List Operations

    Find Data in a Sorted Array O(lg n)
    List只能linear search
    Unsorted Array: O(n)
    sorted Array: O(lg n)
    List: O(n)

    Insert After / Delete After
    Array: O(n)
    List: O(1) ———– 对于这个我只能理解我访问不算做1个operation

    1.5 Queue (Data Structure)

    ADT is not an implementation, it is a description

    Queue: First-in first-out data structure

    • create
    • push
    • pop
    • empty

    An array or a doubly-linked list, O(1) running time
    doubly-linked就是多了【尾指针】+【所有向后的指针】

    1.6 Stack (Data Structure)

    Stack: Last-in first-out data structure

    • create
    • push
    • pop
    • empty

    Singly-linked list is sufficient
    An array or a Singly-linked list, O(1) running time

    Week1 Programming Assignment

    写insertOrdered方法时，对于doubly-linked
    对于插入在第一个位置时，要用 head_ = new_node 

      //1. new a node
  Node *node_new = new Node(newData);
  //2. find the position of newData
  Node *thru = head_;


  /*
  使用 *thru = head_ 时，当我们把 head_ = new_node
  此时head_的地址变化，thru成为了dangling pointer（野指针）
  然后的if语句里可能会设计调用 *thru，此时就会报错 
  [1]    45248 segmentation fault  //报错信息
  */

    总结：编写class的函数时，一定要把给定的class的结构好好看清楚再开始写（比如有哪些private_)
